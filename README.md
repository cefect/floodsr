# floodsr

Super-Resolution for flood hazard rasters.

## Workspace layout

- `.devcontainer/main`: primary development container profile.
- `.devcontainer/docs`: docs-focused container profile.
- `container/`: primary runtime/development Dockerfile and dependencies.
- `docs/`: Sphinx + Read the Docs source and docs-specific container assets.

## Container entry points

- Runtime/code environment: open repo root and choose `.devcontainer/main`.
- Documentation environment: open repo root and choose `.devcontainer/docs`.
